# Pharos Power Skill

Use this skill to analyze Pharos wallets, inspect developer-oriented contract flows, or prepare DeFi-oriented follow-up actions.

Defaults:
- read-first
- investor lane if the prompt does not clearly request developer or DeFi behavior
- no transaction execution without explicit opt-in

Examples:
- Analyze these wallets on Pharos testnet and show me portfolio overlap: 0x...
- Inspect this contract flow and help me scaffold a medium-complexity Pharos dapp
- Review these wallets for DeFi activity and prepare a safe top-up action
