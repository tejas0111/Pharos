03 / Workflow

Deploy in 3 Commands

From zero to verified contract in under 60 seconds. No config files, no boilerplate.

$ forge build
// Compiles Solidity → ABI + bytecode

$ forge create src/Counter.sol:Counter \
  --rpc-url https://rpc.atlantic.pharos.network \
  --private-key $PRIVATE_KEY
// Deploys to Atlantic Testnet

$ npm run demo
// Verifies on-chain via MCP tools
