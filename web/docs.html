01 / Setup

Install Pharos ADS

Clone the repo, install deps, configure your environment. Requires Node.js ≥18 and Foundry for contract compilation.

Clone & Install

$ git clone https://github.com/AnomalyCo/Pharos.git
$ cd Pharos
$ npm install

Set Environment

Your private key is read natively by forge and viem — never logged or exposed.

$ export PRIVATE_KEY=0x...
$ export RPC_URL=https://rpc.atlantic.pharos.network

Verify Installation

$ node --version
v22.0.0
$ forge --version
forge 1.0.0
$ npm run demo
✓ MCP Server started — 18 tools registered
Don't have a testnet wallet? Use the Pharos Atlantic Faucet to get free PHRS.

API Reference

Complete parameter schemas for all 18 MCP tools. * = required.

balanceOf

ParamTypeDefaultDescription
address *string—Address to query
blockTagstringlatestBlock tag (latest, safe, finalized)

transfer

ParamTypeDefaultDescription
to *string—Recipient address
amount *string—PHRS amount (ether unit)

readContract

ParamTypeDefaultDescription
address *string—Contract address
abi *json—Contract ABI array
functionName *string—Function to call
argsarray[]Function arguments
blockTagstringlatestValidated block tag

writeContract

ParamTypeDefaultDescription
address *string—Contract address
abi *json—Contract ABI array
functionName *string—Function to call
argsarray[]Function arguments
valuestring0PHRS value to send

deployContract

ParamTypeDefaultDescription
abi *json—Contract ABI
bytecode *string—Deployable bytecode (0x-prefixed)
argsarray[]Constructor arguments

getLogs

ParamTypeDefaultDescription
addressstring—Contract address filter
fromBlockbigint0Start block (max 100 range)
toBlockbigintlatestEnd block
topicsarray[]Event topic filters

Other Tools

getAccount — address*, blockTag — Pharos-specific RPC (returns balance, nonce, codeHash, storageRoot)

traceTx — txHash*, tracer (callTracer) — debug_traceTransaction (publicly enabled)

getBlock — blockTag* — block number, hash, or tag

getTx — txHash* — transaction receipt + details

getCode — address* — deployed bytecode

getStorage — address* — storage slot (default 0)

sendTx — to*, data, value — raw transaction submission

verifyContract — address*, source*, compilerVersion* — Blockscout verification

estimateGas — to, data, value — gas estimation

getNonce — address* — transaction count

chainId — no params — returns current chain ID

getFees — no params — returns current gas fees

Important Limits

  • No 2300 gas stipend on PHRS transfers — use pull-over-push
  • RPC: ~30 req/s sustained, ~100 req/s burst
  • eth_getLogs max 100 block range per request
  • safe and finalized block tags for production reads
  • eth_getAccount is Pharos-specific (balance, nonce, codeHash, storageRoot)
  • debug_traceTransaction publicly enabled
  • Blockscout verify — no API key on testnet
Chain ID 688689 (NOT 688688). Double-check before deploying — this is the most common mistake.
