01 / Setup

Install Pharos ADS

Clone the repo, install deps, configure your environment. Requires Node.js ≥18 + Foundry.

$ git clone https://github.com/AnomalyCo/Pharos.git
$ cd Pharos
$ npm install
$ export PRIVATE_KEY=0x...
$ export RPC_URL=https://rpc.atlantic.pharos.network

Verify: npm run demo — 18 tools should register.

No testnet PHRS? Use the Atlantic Faucet.

System Architecture
