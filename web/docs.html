01 / Setup

Install

Clone and install. Requires Node.js ≥18 and Foundry.

$ git clone https://github.com/AnomalyCo/Pharos.git
$ cd Pharos
$ npm install
$ export PRIVATE_KEY=0x...
$ export RPC_URL=https://rpc.atlantic.pharos.network
PRIVATE_KEY is read by forge and viem — never printed or committed.

06 / Architecture

Architecture

Dual Cascade. Two layers bridging AI to blockchain:

1. Skill Cascade — 42 subskills in skill/subskills/. Each is markdown with context, examples, conventions. Agents read these to understand Pharos patterns (pull-over-push, no 2300 gas stipend).

2. Tool Cascade — 18 MCP tools in mcp-server/index.js. Each wraps a viem RPC call with validation, error handling, timeouts.

┌──────────┐   ┌──────────┐   ┌──────────┐
│  Agent   │──→│ Subskills │──→│   MCP    │
│(Claude/  │   │ (42 .md) │   │  Server  │
│ Gemini)  │←──│          │←──│ 18 tools │
└──────────┘   └──────────┘   └────┬─────┘
                                   │
                          ┌────────▼──────┐
                          │ Pharos Chain  │
                          │ RPC + viem   │
                          └───────────────┘

Networks

Atlantic Testnet688689rpc.atlantic.pharos.networkexplorer →

Limits

  • — No 2300 gas stipend on PHRS transfers — use pull-over-push
  • — RPC: ~30 req/s sustained, ~100 req/s burst
  • — eth_getLogs: max 100 block range per request
  • — safe, finalized tags available for production reads
  • — eth_getAccount is Pharos-specific
  • — debug_traceTransaction publicly enabled
  • — Blockscout verify — no API key on testnet
Chain ID 688689 (NOT 688688). Double-check before deploying.
