# Pharos Power Skill

Pharos Power Skill is a natural-language-first onchain intelligence skill for investors, developers, and DeFi power users on Pharos.

## Features

- investor lane with multi-wallet aggregation
- developer lane with contract and integration guidance
- DeFi lane with approval and protocol signals
- safe and expert execution gating

## Setup

```bash
npm install
```

Set at least one RPC URL depending on the network you want to use:

- `PHAROS_TESTNET_RPC_URL`
- `PHAROS_MAINNET_RPC_URL`

## Commands

```bash
npm test
npm run build
```

## Example prompts

- Analyze these 3 wallets on Pharos mainnet and summarize overlap
- Inspect this contract flow and prepare a dapp interaction plan
- Review DeFi protocol overlap and suggest next actions
