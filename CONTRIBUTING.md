# Contributing

Thanks for your interest in the Pharos Agent Dev Suite!

## How to Contribute

1. **Fork** the repo and create a feature branch.
2. **Make your changes** — follow the existing code style and conventions.
3. **Run tests**:
   ```bash
   npm test
   ```
   This runs both Solidity tests (`forge test`) and MCP server tests (`node mcp-server/test.js`).
4. **Format Solidity**: `forge fmt`
5. **Open a PR** with a clear title and description.

## What We're Looking For

- Bug fixes and security improvements
- Additional Pharos-specific subskills
- Better test coverage
- Documentation improvements
- New MCP tool ideas

## Code of Conduct

Please be respectful and constructive. See `CODE_OF_CONDUCT.md`.
